## Why

Maestro 项目当前缺少项目标志性视觉形象和完善的文档。项目名称 "Maestro"（指挥家）天然适合用"指挥家与乐队"的视觉隐喻来表达其核心理念——一位人类开发者指挥多个 AI Agent 协同工作。同时项目需要中英双语的 README 文档来面向不同语言的开发者群体。

## What Changes

- 新增项目概念艺术图：人类指挥家在舞台上指挥，台下是一群小龙虾（Claw Agent）在演奏乐器，体现 Maestro 的 Human-to-Agents 协作模型
- 新增英文 README（`README.md`）：项目介绍、核心特性、安装使用、架构概览
- 新增中文 README（`README_CN.md`）：与英文版内容对应的中文文档

## Capabilities

### New Capabilities
- `project-artwork`: 项目概念艺术图的生成规格，包括风格、构图、隐喻元素的定义
- `readme-docs`: 中英双语 README 文档的内容结构和写作规范

### Modified Capabilities
_无_

## Impact

- 项目根目录新增文件：`README.md`、`README_CN.md`、项目概念图（PNG 格式）
- 不影响任何现有代码、API 或依赖
- 纯文档/资产类变更，无破坏性影响
